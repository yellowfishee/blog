# 我的笔记
## 介绍
从大一记录笔记到现在已经过了差不多四年，仔细想象印象中的笔记最后都不知道留在了哪个垃圾堆里。为了让自己的笔记更加的高效，并且给自己的学习留出更多的时间。现在来使用emacs+org roam + 优化之后的卡片笔记法来搭建自己的第二大脑。

### 为什么不选择 Obsidian
其实原因很简单，因为我的需求非常简单，但是对于后续的整理，可能我还是会使用 Obsidian 毕竟生态在那个地方。那么对于现在来说，我只需要将自己的输入输出做好一定的整理就好了。


## 工作流
按照项目管理的方式，首先确定好自己需要学习的项目，如：奇门遁甲，那么就会按照卡片笔记法的方式，分为闪念笔记（使用Flomo）、文献笔记（摘抄），以及最后的永久笔记。这个概念和卡片笔记没有太大的区别，只是需要注意定期在Flomo中回顾自己的灵感，让自己的思维不要陷入枯竭的地步。
